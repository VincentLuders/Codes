
# Text Expander Application

This is a text expansion application with a searchable interface and system tray functionality.

## Setup
1. Install Python 3.7 or higher
2. Install the required packages: `pip install -r requirements.txt`
3. Ensure AutoHotkey is installed (the script will attempt to find it automatically)
4. Run the main application: `python main.py`

## Usage
- Press Ctrl+I to open the search interface
- Type to search for snippets
- Press Esc or click outside the search box to hide the interface
- The application will run in the system tray

## Customizing Snippets
Edit the `text_expander.ahk` file to add or modify text expansion snippets.
